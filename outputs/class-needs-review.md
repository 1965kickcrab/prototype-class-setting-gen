# 클래스 기능 명세 - 정책 확인 필요

## 정책 확인 필요 항목

1. 일자별 예약 상세의 클래스 표시 방식
   - 확인된 사실: 일자별 예약 상세는 같은 날짜의 예약을 모두 표시하며, 현재 각 반려견 카드에 클래스명을 표시한다. 예약 이력 상세의 `예약 정보`는 클래스 항목을 제공한다.
   - 결정이 필요한 정책: 일자별 예약 상세의 `예약 정보`에 `클래스` 항목을 추가할 때, 서로 다른 클래스가 함께 예약된 경우의 표시 방식을 정해야 한다.
   - 추천 기본값: 예약 정보에는 `클래스` 항목을 추가하지 않고 반려견별 카드에 클래스명을 표시한다. 단일 클래스만 존재하는 예약 그룹 상세에는 `클래스` 항목에 해당 클래스명을 표시한다.
